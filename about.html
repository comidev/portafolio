¡Hola, Soy Omar Miranda!

Full Stack Java & React Developer Junior

Soy un ex estudiante de la carrera de Ingeniería de Sistemas de la Universidad Nacional Mayor de San Marcos (UNMSM), Lima, Perú.

A partir de documentaciones y plataformas online, comprendo sobre Microservicios nivel básico con Spring Cloud, Config Server, Eureka, Feign, Hystrix y API Gateway.

Por último, tengo conocimiento Frontend Junior de React, JavaScript, ES6+, HTML5, CSS3+, Emotion, StyledComponent y Responsive Design.
